# Judge Flow Checklist

- DAO creates mandate.
- Agent registers with persistent identity.
- Agent posts bond.
- Compliant action passes.
- Trace is generated and hashed.
- Trace is stored.
- Trace root is committed.
- Prompt-injected action is proposed.
- ActionGate rejects it.
- Bond is slashed.
- Compliance score drops.
- Violation trace is replayable.
- Tamper detector catches mutation.
